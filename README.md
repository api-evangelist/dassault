# Dassault Systèmes

Dassault Systèmes provides the 3DEXPERIENCE platform with thousands of publicly available REST APIs for product design, simulation, and manufacturing industries. The platform covers SolidWorks, CATIA, ENOVIA, and 3D collaboration tools, exposing web services across engineering items, manufacturing processes, CAD collaboration, project tasks, and partner order management through standardized RESTful endpoints.
